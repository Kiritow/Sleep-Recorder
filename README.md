# Sleep-Recorder  

## Used Encoding Methods  
**HC General Encode Method ( HC TECH )**  
FILE --> FILE.enc  
FILE.enc --> FILE  
**HC TECH Internal Mix Encoding Method ( HC TECH )**  
FILE --> FILE.mix  
FILE.mix --> FILE  
**HC TECH Int/Char Converter**  
FILE --> FILE.int  
FILE.int --> FILE
